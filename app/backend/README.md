## API Documentation

The backend provides the following APIs:

### GET /api/users

Returns a list of all users.

### GET /api/users/{id}

Returns the details of a specific user.

### POST /api/users

Creates a new user.

### PUT /api/users/{id}

Updates the details of a specific user.

### DELETE /api/users/{id}

Deletes a specific user.
